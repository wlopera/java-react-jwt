# java-react-jwt
Proyecto Java - React - Uso Token - Login - Logout - Data

Java:
  * JWT
  * Spring Boot
  * Security
  * Servicios Rest que permite:
    * Login: user/pass => Genera token
    * Consulta de Paises ==> Debe enviarse Token (filtro para validar token)

React:
  * APP básica de React
    * Navegación
    * Estilos css
    * Login
    * Consultar Países
    * Logout
